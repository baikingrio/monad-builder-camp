# Monad Builder Camp 学习计划

## 总目标

在 Web3 暑期实习计划 - Monad Builder Camp 中，用 AI 辅助完成一个 Monad 生态链上产品，从学习记录、原型实验、Hackathon 提交到作品集整理形成完整 Proof of Work。

## 每日节奏

### 最小路径（忙的时候）

- 阅读 / 学习 30–45 分钟
- 做一个小记录：概念、交易、工具、项目想法任选一个
- 写 100–200 字残酷共学打卡

### 推荐路径

- 学习 Monad / Web3 / AI coding 60–90 分钟
- 做一个小实验或项目推进 60–120 分钟
- 记录 daily note + proof link + 下一步

### 挑战路径

- 每天围绕最终项目推进一个明确交付：合约、前端、链上交互、文档、Demo、视频脚本或 Pitch Deck

## Week 1：共同底座 — Enter Onchain World

目标：完成课程钱包、链上交互、Monad 生态和 AI 工具准备。

产出建议：

- 课程钱包和测试网交互记录
- Monad 生态基础笔记
- 第一次交易 / 合约交互记录
- AI 辅助学习 workflow
- 每日公开学习笔记

## Week 2：Builder Track Split

目标：选择 Tech Track 为主，Research / Product 为辅。

产出建议：

- Monad 项目方向候选清单
- 1–2 个小型链上实验
- MVP Scope 草稿
- 技术选型与用户场景说明
- Proof of Work 索引

## Week 3：Builder Collaboration & Mini Demo Week

目标：围绕一个产品想法完成 Prototype，并准备 Mini Demo。

产出建议：

- 项目 README 初版
- Demo Story
- 前端 / 合约 / 链上交互原型
- 用户场景与传播叙事
- Mini Demo 讲稿和反馈记录

## Week 4：Monad Builder Season & Hackathon Week

目标：从 Prototype 到 Product，完成可运行、可展示、可提交的 Monad 链上产品。

最终交付：

- 可运行产品链接或 Repo
- 合约地址、Transaction Hash、Explorer 链接
- Pitch Deck
- 2–5 分钟 Demo Video
- Final Presentation

## Week 5：作品集与简历 Workshop

目标：把训练营成果整理成职业资产。

产出建议：

- Portfolio 项目页
- GitHub / Demo / Pitch Deck / 视频链接索引
- 简历 Bullet Point
- 个人学习日志精选
- 研究或运营案例整理

## 每周复盘问题

1. 本周我真正做出来了什么？
2. 哪些内容可以作为公开 Proof of Work？
3. 哪些项目想法应该继续保持私密？
4. 下周最重要的一个交付是什么？
5. 我的作品集材料新增了什么？
